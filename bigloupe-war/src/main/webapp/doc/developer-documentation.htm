Developer documentation

This documentation helps you to develop BigLoupe. Apache Maven (Makefile in Java) is highly used.

  • Package
    • Package map-reduce-job-monitoring

Package bigloupe-map-reduce-job-monitoring

Create ZIP file with all dependencies. This ZIP file must be decompacted in your $PIG_HOME/lib directory. You could follow pig scripts easily.

mvn install -Pzip : build a zip (in bigloupe-map-reduce-job-monitoring/target/bigloupe-map-reduce-job-monitoring-${version}-bin.zip)

Package chart engine

Create JAR file with all dependencies for BigLoupe chart.

mvn install : build a jar (in bigloupe-chart/target/bigloupe-chart-${version}-bin.zip)
